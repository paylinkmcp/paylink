Get PayLink up and running in your development environment in minutes.

## <Icon icon="download" size={20} /> Install

<CodeGroup>
  ```bash uv
  uv add paylink
  # Requires Python 3.9+
  ```

  ```bash pip
  pip install paylink
  # Requires Python 3.9+
  ```
</CodeGroup>
