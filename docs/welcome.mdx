## Welcome to PayLink

Thanks for checking out PayLink! This guide gives you the lay of the land before you jump into installation or the quickstart.

## How the Docs Are Organized

- **Introduction**: High-level overview of PayLink and what problems it solves.
- **Install**: Step-by-step setup instructions for your local environment.
- **Quickstart**: Build and run your first workflow end-to-end.
- **Examples**: Realistic scenarios powered by MCP integrations.

Each page includes copyable snippets, diagrams, and pointers to supporting resources.

## Prerequisites

- Node.js 18+ and pnpm installed locally.
- Access to at least one supported provider account (e.g., M-Pesa, Airtel Money).
- MCP-compatible runtime or agent to execute the tools.

If you are missing any of these items, the `Install` page links to handy setup guides.

## Recommended Next Steps

1. Skim the `Introduction` to understand PayLink's value prop.
2. Follow the `Install` guide to configure your environment.
3. Walk through the `Quickstart` to validate your setup.
